# Pokemon Client

Simple Python application that gets Pokemon information using the PokeAPI

## Features

- Search Pokemon by name
- You will get:
  - Name
  - Height
  - Weight

## Installation

Install uv:

```bash
pip install uv
```

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/pokemon-client.git
```

Go to project folder:

```bash
cd pokemon-client
```

Install dependencies:

```bash
uv sync
```

## Run the application

```bash
uv run main.py
```

## Example

```text
Enter pokemon name: pikachu

Pokemon Information
-------------------
Name: pikachu
Height: 4
Weight: 60
```
